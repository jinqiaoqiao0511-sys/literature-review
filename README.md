## 目录结构

```
每周文献调研/
├── 文献调研/                  ← 最终成果
│   ├── Weekly/               ← 每周调研摘要（Markdown）
│   ├── 2026-WXX/             ← 高优先级论文 Obsidian 笔记
│   └── 文献调研记录_YYYY-WXX.xlsx  ← Excel 汇总表
├── 中间结果/                  ← 自动化流水线中间产物
│   ├── step01a/b-merged-*.md ← 检索结果
│   ├── step02a/b-merged-*.md ← 扩展检索结果
│   ├── step03-classified-*.md← 整合分类+评级（核心输出）
│   └── journal-cache/        ← 期刊 JCR/IF 缓存（跨周复用）
└── 自动化就绪版0428/          ← 13 个自动化任务的 Prompt 定义
    ├── 00-总控文档.md             ← 架构说明与字段流转
    └── auto-00 ~ auto-07         ← 各步骤 Prompt（含新增任务）
```

## 自动化流水线

每周日 23:00 至周一 11:00 自动执行 13 个任务，覆盖从预检到知识库输出的完整流程：

```
周日 23:00  auto-00   预检（判断是否执行检索）     ← 新增
周一 00:00  auto-01a  检索通用AI                   ──┐
周一 01:00  auto-01b  检索启动子发酵                ──┤
周一 02:00  auto-02a  检索IDP-生物智能体            ──┼─→ step01/02
周一 03:00  auto-02b  检索LLM-专项                 ──┘
周一 04:00  auto-03a  数据整合+相关性校验           ──→ step03-merged
周一 05:00  auto-03b  分类评级                    ──→ step03-classified
周一 06:00  auto-03c  更新期刊缓存                 ──→ journal-cache
周一 07:00  auto-04a  Excel输出                   ──→ .xlsx
周一 08:00  auto-04b  Obsidian周报+笔记           ──→ .md
周一 09:00  auto-05   巡检校验+标注               ──→ inspection report
周一 10:00  auto-06   文献深度阅读（调用paper-quick-reader）← 新增
周一 11:00  auto-07   知识库问答准备              ──→ knowledge-base  ← 新增
```

## 模型分配

| 模型 | 任务 | 理由 |
|------|------|------|
| GLM-5.1 | 检索 (01a/01b) + 预检 (00) | 工具调用稳定性高 |
| MiniMax-2.7 | 扩展检索 (02a/02b) + 深度阅读 (06) | 语义扩展能力强 |
| Kimi-K2.6 | 整合/输出 (03~04) + 巡检 (05) + 知识库 (07) | 长上下文 + 指令遵循精确 |

## 调用技能

| 技能 | 用途 | 调用任务 |
|------|------|----------|
| 文献相关性校验 | 对文献进行相关性评分和分类 | auto-03a |
| 文献标注 | 调用 paper-quick-reader 进行批量标注 | auto-05 |
| paper-quick-reader | 文献深度阅读（三档：裸读/淫道/精读） | auto-06 |

## 模型分配

| 模型 | 任务 | 理由 |
|------|------|------|
| GLM-5.1 | 检索 (01a/01b) | 工具调用稳定性高 |
| MiniMax-2.7 | 扩展检索 (02a/02b) + 巡检 (05) | 语义扩展能力强 |
| Kimi-K2.6 | 整合/输出 (03~04) | 长上下文 + 指令遵循精确 |

## 如何使用

1. **查看本周调研**：打开 `文献调研/Weekly/2026-WXX-调研摘要.md`
2. **查看 Excel 汇总**：打开 `文献调研/文献调研记录_2026-WXX.xlsx`
3. **查看高优论文笔记**：浏览 `文献调研/2026-WXX/` 目录
4. **查看原始分类数据**：打开 `中间结果/step03-classified-2026-WXX.md`
