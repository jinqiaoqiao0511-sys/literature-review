# 每周文献调研 — 自动化文献检索与管理

基于 Obsidian + WorkBuddy 自动化的每周学术文献调研系统，覆盖合成生物学、AI for Science、发酵工程等研究方向。

## 研究方向

| 方向 | 关键词范围 | 说明 |
|------|-----------|------|
| 通用 AI | #1-12 | AI/ML 在生物领域的通用应用 |
| 启动子与发酵 | #13-25 | 启动子工程、发酵优化 |
| IDP 与生物智能体 | #26-33 | 内在无序蛋白、生物智能体 |
| LLM 专项 | #34-41 | 大语言模型在生命科学中的应用 |

## 目录结构

```
每周文献调研/
├── 文献调研/                  ← 最终成果
│   ├── Weekly/               ← 每周调研摘要（Markdown）
│   ├── 2026-WXX/             ← 高优先级论文 Obsidian 笔记
│   └── 文献调研记录_YYYY-WXX.xlsx  ← Excel 汇总表
├── 中间结果/                  ← 自动化流水线中间产物
│   ├── step01a/b-merged-*.md ← 检索结果
│   ├── step02a/b-merged-*.md ← 扩展检索结果
│   ├── step03-classified-*.md← 整合分类+评级（核心输出）
│   └── journal-cache/        ← 期刊 JCR/IF 缓存（跨周复用）
└── 自动化就绪版0428/          ← 10 个自动化任务的 Prompt 定义
    ├── 00-总控文档.md         ← 架构说明与字段流转
    └── auto-01a ~ auto-05    ← 各步骤 Prompt
```

## 自动化流水线

每周一自动执行 10 个任务，覆盖从检索到输出的完整流程：

```
00:00  auto-01a  检索通用AI          ──┐
01:00  auto-01b  检索启动子发酵       ──┤
02:00  auto-02a  检索IDP-生物智能体   ──┼─→ step01/02
03:00  auto-02b  检索LLM-专项        ──┘
04:00  auto-03a  数据整合             ──→ step03-classified
05:00  auto-03b  分类评级             ──→ 优先级/方向标注
06:00  auto-03c  更新期刊缓存         ──→ journal-cache
07:00  auto-04a  Excel输出            ──→ .xlsx
08:00  auto-04b  Obsidian周报+笔记    ──→ .md
09:00  auto-05   巡检校验+自动修复    ──→ inspection report
```

## 模型分配

| 模型 | 任务 | 理由 |
|------|------|------|
| GLM-5.1 | 检索 (01a/01b) | 工具调用稳定性高 |
| MiniMax-2.7 | 扩展检索 (02a/02b) + 巡检 (05) | 语义扩展能力强 |
| Kimi-K2.6 | 整合/输出 (03~04) | 长上下文 + 指令遵循精确 |

## 如何使用

1. **查看本周调研**：打开 `文献调研/Weekly/2026-WXX-调研摘要.md`
2. **查看 Excel 汇总**：打开 `文献调研/文献调研记录_2026-WXX.xlsx`
3. **查看高优论文笔记**：浏览 `文献调研/2026-WXX/` 目录
4. **查看原始分类数据**：打开 `中间结果/step03-classified-2026-WXX.md`
