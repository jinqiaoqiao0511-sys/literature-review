> **输出**：
> - 更新 `D:\.A硕士\.A干\每周文献调研(1)\每周文献调研\journal-cache\` 文件夹（根目录）

## 第一步：计算ISO周编号

请根据当前日期计算ISO周编号：
- 上周一 = 今天往前推到今天所在的周一，再往前推7天
- 确定上周的ISO周编号：YYYY-WXX

---

## 第二步：检查输入文件

检查 `D:\.A硕士\.A干\每周文献调研(1)\每周文献调研\中间结果\step03-journals-YYYY-WXX.md` 是否存在：

- **若文件存在** → 读取文件内容，解析期刊缓存更新列表
- **若文件不存在** → 停止执行，输出错误信息：
  "step03-journals文件不存在，请先执行 auto-03b（分类评级与输出）任务。"

---

## 第三步：读取现有缓存

读取 `D:\.A硕士\.A干\每周文献调研(1)\每周文献调研\journal-cache\` 文件夹：

- 若文件夹存在 → 遍历文件夹，记录已缓存的DOI列表
- 若文件夹不存在 → 创建该文件夹，并记录为空缓存

---

## 第四步：解析待缓存期刊

从 step03-journals-YYYY-WXX.md 解析期刊列表，格式：

```markdown
| 期刊名 | JCR分区 | 影响因子 | DOI | 来源 |
|--------|---------|----------|-----|------|
```

对于每个期刊记录：
1. 提取 DOI 编号
2. 检查该 DOI 是否已在缓存中存在
3. 若已缓存 → 跳过（不覆盖已有记录）
4. 若未缓存 → 准备创建新缓存文件

---

## 第五步：创建/更新缓存文件

对于每个需要缓存的期刊（DOI未在缓存中存在）：

**文件夹路径**：`D:\.A硕士\.A干\每周文献调研(1)\每周文献调研\journal-cache\`

**文件名规则**：
- 将DOI中的 `/` 替换为 `_`，`.` 替换为 `_`
- 例如：`10.1038/nbt.1234` → `10_1038_nbt_1234.md`
- 例如：`10.1016/j.molbio.5678` → `10_1016_j_molbio_5678.md`

**单个文件格式**：
```markdown
# 期刊缓存 - [DOI]

| 字段 | 值 |
|------|---|
| 期刊名 | [期刊名] |
| JCR分区 | [Q1/Q2/Q3/Q4] |
| 影响因子 | [数值] |
| 更新日期 | [YYYY-MM-DD] |
```

**写入后确认**：
- 文件成功写入
- 文件大小 > 0
- 格式正确
