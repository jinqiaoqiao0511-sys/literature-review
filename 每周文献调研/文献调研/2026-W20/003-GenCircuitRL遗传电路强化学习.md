# Core Finding

GenCircuit-RL uses reinforcement learning with hierarchical verification rewards and a 4-stage curriculum to generate SBOL3 code for genetic circuit design, improving functional reasoning by 14-16 percentage points over binary rewards.

# Abstract

Genetic circuit design remains a laborious expert-driven process. GenCircuit-RL produces Python code in pysbol3 to construct genetic circuits in SBOL, with hierarchical verification rewards and four-stage curriculum achieving +14 to +16 percentage points improvement over binary rewards on functional reasoning tasks.
